Search Tips

Based on user feedback, we have compiled the following tips.
  • Check the spelling of the city name.
If you are not sure of the spelling of the city, use the state (OH or Ohio) or country name (Mexico) and drill down to the city.
  • When you use two letter state codes, do not add periods to the code.
    Works - Lexington NC
    Doesn't Work - Lexington N.C.
  • Do not search using the city and the zip code.
    Works - Atlanta or 30346
    Doesn't Work - Atlanta 30346
  • Do not use 4 letter state codes. Use 2 letter codes.
    Works - Cedar Hill TN
    Doesn't Work - Cedar Hill TENN
  • Do not use the country England. weather.com list cities in England under United Kingdom.
    Works - London or London, United Kingdom
    Doesn't Work - London England
  • Do not use ISO country codes. These are not supported.
    Works - Sydney or Sydney Australia
    Doesn't Work - Sydney AU
